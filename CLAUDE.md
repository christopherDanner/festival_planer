# Festival Planer

Musikverein Personaleinteilung – Webtool zur Verteilung von Vereinsmitgliedern auf Feststationen.

## Tech Stack

- Python 3, Streamlit, pandas, openpyxl

## Befehle

- Installation: `pip install -r requirements.txt`
- App starten: `streamlit run app.py`

## Konventionen

- Sprache: Deutsch (Code-Kommentare, UI-Texte, Commit-Messages)
- Flat-Struktur, kein Klassenmodell – Streamlit-Script-Stil
- 4 Spaces Einrückung (Python-Standard)
